# Equadores-project


====================================================================

SOLUCIONADOR DE EQUACOES DE SEGUNDO GRAU
====================================================================

FUNCIONALIDADES IMPLEMENTADAS:
-------------------------------
1. Resolucao completa de equacoes ax² + bx + c = 0
2. Grafico interativo da parabola
3. Validacao robusta de coeficientes
4. Historico de equacoes resolvidas
5. Modo claro/escuro
6. Design totalmente responsivo
7. Conteudo educativo interativo
8. Sistema de notificacoes
9. Explicacao passo a passo dos calculos

COMO USAR:
----------
1. Abra o arquivo index.html em um navegador web
2. Insira os coeficientes a, b e c nos campos indicados
3. Clique em "Calcular Raizes"
4. Visualize os resultados, grafico e explicacoes
5. Use o historico para revisar equacoes anteriores
6. Alternar tema usando o botao sol/lua

TIPOS DE RESULTADO:
-------------------
- Duas raizes reais distintas (Delta > 0)
- Uma raiz real dupla (Delta = 0)
- Raizes complexas (Delta < 0)

TECNOLOGIAS:
------------
- HTML5, CSS3, JavaScript ES6+
- Chart.js para graficos
- Google Fonts (Inter + JetBrains Mono)
- LocalStorage para persistencia
- Design responsivo com CSS Grid e Flexbox

EXEMPLOS DE USO:
----------------
Entrada: a=1, b=-5, c=6
Resultado: x₁=3, x₂=2 (duas raizes reais)

Entrada: a=1, b=-4, c=4  
Resultado: x=2 (raiz dupla)

Entrada: a=1, b=0, c=1
Resultado: x₁= i, x₂=-i (raizes complexas)
